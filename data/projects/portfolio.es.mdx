Sitio web personal creado para mostrar mis proyectos, habilidades y para sirvir de medio para interactuar con clientes y
otros programadores.

También es usado como el lugar donde probar lo que voy aprendiendo sobre desarrollo web, web3 y donde publico algunos artículos como medio
de compartir mis (re)descubrimientos, organizar mis ideas y tener un lugar al que volver por contenido sistematizado sobre algún tema de mi
interés.

Desarrollado inicialmente con HTML5, TailwindCSS y algo de JavaScript.

Actualizado a Next.js para incluir un blog, mejorar SEO, usar componentes y facilitar la reutilización de software.

TailwindCSS fue una herramienta muy útil, a la hora de seguir un sistema de diseño y crear un sitio web a la medida, gracias a su
característica utilidad-primero (utility-first), que permite iteraciones y actualizaciones rápidas de prototipos.

El proyecto fue diseñado para proveer una buena experiencia de usuario y tener buen rendimiento. Gimp y imageoptim fueron herramientas
utilizadas para crear imágenes de acuerdo a distintos tamaños de pantalla y para su compresión en busca de mejorar el rendimiento.

Aunque inicialmente usé Lighthouse (Google Chrome Console Tool) y RespImageLint para detectar problemas en las imágenes que afectan el
rendimiento, luego de la transición a Next.js, la mayor parte de este trabajo es realizado por el framework y el componente Image.
