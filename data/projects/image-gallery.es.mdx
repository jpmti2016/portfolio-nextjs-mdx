En este proyecto usé CSS Grid para construir un sitio web flexible. Se generan imágenes de diferentes tamaños de acuerdo a la celda del
`grid` que las contiene. Esta técnica es del curso CSS Grid de Wes Bos.

Usé **Gimp** para crear imágenes de varias dimensiones. Usé **imageoptim** para comprimir y optimizar imágenes para mejorar el rendimiento
del sitio web.

También sirvió de práctica de trabajo con otras herramientas de procesamiento y análisis de imágenes como Lighthouse y RespImageLint.
