Amazon Web Services (AWS), entre otros proveedores de computación en la nube, habilita a nuestros equipos para que se enfoquen en la creación
de valor para el cliente. No tener que dedicar tiempo y recursos al manejo de la infraestructura significa que podremos disponerlos para el
logro de los objetivos del negocio.

Un sistema de precios flexible donde sólo pagas por los recursos que has usado y una generosa capa gratuita ponen AWS al alcance de muchos
presupuestos.

Con AWS Amplify, desde la etapa de prototipos hasta el despliegue, puedes usar el esquema GraphQL como guía del ciclo de vida del desarrollo
de software. Tienes disponible para ello un servicio de despliegue completamente gestionado, con varios flujos de trabajo de integración y
despliegue creados para responder a tus necesidades.

Desde el diseño de la API hasta la etapa de producción puedes usar varios servicios de AWS como bloques de construcción para tu aplicación.
Si necesitas autenticación / autorización, Cognito viene en tu ayuda. Si necesitas incluir capacidades `offline`, tienes a AWS DataStore para ello...
AWS AppSync disminuye la fricción entre rápidas iteraciones de tu GraphQL api.

AWS Amplify provee todo lo necesario para el desarrollo de aplicaciones web / móviles, incluyendo servicios para API (GraphQL o Rest),
autenticación, inteligencia artificial / aprendizaje automático, integración continua / despliegue continuo y funciones Lambda, por
sólo mencionar unos pocos. Todo desde la comodidad de tu terminal con una interfaz de línea de comandos (CLI) o desde Amplify Studio si
prefieres una interfaz visual.

Para facilitar tu trabajo cuentas con varias librerías creadas para tu plataforma objetivo: JavaScript, Android, Flutter o iOS. También
incluye componentes para ReactJS, Angular, Vue, Componentes Web y React Native.

Este proyecto fue construido mayormente con AWS Amplify, GraphQL y ReactJS. Para probar la aplicación web puedes registrarte
o usar `usuario: test, contraseña: test2020`.
